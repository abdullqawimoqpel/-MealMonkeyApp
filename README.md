# Meal Monkey App 🐒🍔

تطبيق توصيل طعام (Food Delivery App) متكامل مبني باستخدام إطار عمل فلاتر (Flutter). 

## 🌟 مميزات المشروع (Features)
- واجهة مستخدم عصرية وجذابة تعتمد على Material Design.
- دعم اللغتين العربية والإنجليزية بشكل مدمج، مع تعيين العربية كلغة رئيسية وافتراضية.
- إدارة سلسة لحالة التطبيق باستخدام `Provider` (لإدارة عربة التسوق، المطاعم، وإعدادات التطبيق).
- استخدام خط (Cairo) لدعم القراءة بشكل أنيق باللغة العربية.
- شاشة افتتاحية (Splash Screen) مميزة.

## 📦 المكاتب و الحزم (Dependencies)
المشروع يعتمد بشكل أساسي على الحزم التالية:
- `provider`: لإدارة الحالة (State Management).
- `http`: للتعامل مع واجهات برمجة التطبيقات (API Requests).
- `shared_preferences`: لحفظ إعدادات المستخدم والبيانات محلياً.
- `cached_network_image`: لتحميل الصور من شبكة الإنترنت بطريقة سريعة مع حفظها مؤقتاً.

## 🚀 التشغيل المبدئي (Getting Started)

تأكد من تنصيب بيئة فلاتر (Flutter) على جهازك، ثم قم بتشغيل الأوامر التالية من سطر الأوامر الخاص بالمشروع:

```bash
# تثبيت الحزم المطلوبة
flutter pub get

# تشغيل التطبيق في المحاكي أو جهاز حقيقي
flutter run
```
